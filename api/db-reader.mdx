title: DbReader
description: Read-only interface for accessing a SlateDB database

The `DbReader` struct provides a read-only interface for accessing a database from either the latest persistent state or from an arbitrary checkpoint.

## Opening a Reader

### open()

Creates a database reader that can read the contents of a database (but cannot write any data). The caller can provide an optional checkpoint. If the checkpoint is provided, the reader will read using the specified checkpoint and will not periodically refresh the checkpoint. Otherwise, the reader creates a new checkpoint pointing to the current manifest and refreshes it periodically as specified in the options. It also removes the previous checkpoint once any ongoing reads have completed.

```rust
pub async fn open<P: Into<Path>>(
    path: P,
    object_store: Arc<dyn ObjectStore>,
    checkpoint_id: Option<Uuid>,
    options: DbReaderOptions
) -> Result<Self, Error>
```

<ParamField path="path" type="P: Into<Path>" required>
  The path to the database
</ParamField>

<ParamField path="object_store" type="Arc<dyn ObjectStore>" required>
  The object store to use for reading
</ParamField>

<ParamField path="checkpoint_id" type="Option<Uuid>">
  Optional checkpoint ID to read from. If `None`, creates and manages its own checkpoint
</ParamField>

<ParamField path="options" type="DbReaderOptions" required>
  Configuration options for the reader
</ParamField>

<ResponseField name="Result" type="Result<DbReader, Error>">
  Returns the reader instance on success, or an error if the reader could not be opened
</ResponseField>

### builder()

Creates a new builder for a database reader at the given path.

```rust
pub fn builder<P: Into<Path>>(
    path: P,
    object_store: Arc<dyn ObjectStore>
) -> DbReaderBuilder<P>
```

<ParamField path="path" type="P: Into<Path>" required>
  The path to the database
</ParamField>

<ParamField path="object_store" type="Arc<dyn ObjectStore>" required>
  The object store to use for reading
</ParamField>

<ResponseField name="DbReaderBuilder" type="DbReaderBuilder<P>">
  Returns a builder to configure and initialize the reader
</ResponseField>

**Example:**

```rust
use slatedb::{Db, DbReader, Error};
use slatedb::object_store::{ObjectStore, memory::InMemory};
use std::sync::Arc;

#[tokio::main]
async fn main() -> Result<(), Error> {
    let object_store: Arc<dyn ObjectStore> = Arc::new(InMemory::new());
    // First create a database
    let db = Db::open("test_db", Arc::clone(&object_store)).await?;
    db.close().await?;
    // Then open a reader
    let reader = DbReader::builder("test_db", object_store)
        .build()
        .await?;
    Ok(())
}
```

## Read Operations

### get()

Get a value from the database with default read options.

```rust
pub async fn get<K: AsRef<[u8]> + Send>(&self, key: K) -> Result<Option<Bytes>, Error>
```

<ParamField path="key" type="K: AsRef<[u8]> + Send" required>
  The key to get
</ParamField>

<ResponseField name="Result" type="Result<Option<Bytes>, Error>">
  Returns `Some(Bytes)` if the key exists, `None` if it doesn't exist, or an error
</ResponseField>

**Example:**

```rust
use slatedb::{Db, DbReader, Error};
use slatedb::object_store::memory::InMemory;
use std::sync::Arc;

#[tokio::main]
async fn main() -> Result<(), Error> {
    let object_store = Arc::new(InMemory::new());
    
    // Create and write to database
    let db = Db::open("test_db", Arc::clone(&object_store)).await?;
    db.put(b"key", b"value").await?;
    db.close().await?;
    
    // Open reader and read data
    let reader = DbReader::builder("test_db", object_store)
        .build()
        .await?;
    let value = reader.get(b"key").await?;
    assert_eq!(value, Some("value".into()));
    
    Ok(())
}
```

### scan()

Scan a range of keys using the default scan options.

```rust
pub async fn scan<K, T>(&self, range: T) -> Result<DbIterator, Error>
where
    K: AsRef<[u8]> + Send,
    T: RangeBounds<K> + Send
```

<ParamField path="range" type="T: RangeBounds<K> + Send" required>
  The range of keys to scan (e.g., `"a".."z"`, `..`, `"key"..`)
</ParamField>

<ResponseField name="Result" type="Result<DbIterator, Error>">
  Returns an iterator with the results of the scan
</ResponseField>

**Example:**

```rust
use slatedb::{Db, DbReader, Error};
use slatedb::object_store::memory::InMemory;
use std::sync::Arc;

#[tokio::main]
async fn main() -> Result<(), Error> {
    let object_store = Arc::new(InMemory::new());
    
    // Create and write to database
    let db = Db::open("test_db", Arc::clone(&object_store)).await?;
    db.put(b"a", b"a_value").await?;
    db.put(b"b", b"b_value").await?;
    db.close().await?;
    
    // Open reader and scan
    let reader = DbReader::builder("test_db", object_store)
        .build()
        .await?;
    let mut iter = reader.scan("a".."c").await?;
    
    let kv = iter.next().await?.unwrap();
    assert_eq!(kv.key.as_ref(), b"a");
    
    Ok(())
}
```

## Complete Example

Here's a complete example showing how to use a database reader:

```rust
use slatedb::{Db, DbReader, Error};
use slatedb::config::DbReaderOptions;
use slatedb::object_store::memory::InMemory;
use std::sync::Arc;
use std::time::Duration;

#[tokio::main]
async fn main() -> Result<(), Error> {
    let object_store = Arc::new(InMemory::new());
    
    // Create and populate database
    let db = Db::open("test_db", Arc::clone(&object_store)).await?;
    db.put(b"user:1", b"Alice").await?;
    db.put(b"user:2", b"Bob").await?;
    db.put(b"user:3", b"Charlie").await?;
    db.flush().await?;
    db.close().await?;
    
    // Open a reader with custom options
    let reader = DbReader::builder("test_db", object_store)
        .with_options(DbReaderOptions {
            manifest_poll_interval: Duration::from_secs(5),
            ..Default::default()
        })
        .build()
        .await?;
    
    // Read individual values
    let alice = reader.get(b"user:1").await?;
    println!("User 1: {:?}", alice);
    
    // Scan with prefix
    let mut iter = reader.scan_prefix(b"user:").await?;
    while let Some(kv) = iter.next().await? {
        println!("Key: {:?}, Value: {:?}", 
            String::from_utf8_lossy(&kv.key),
            String::from_utf8_lossy(&kv.value)
        );
    }
    
    reader.close().await?;
    Ok(())
}
```

## DbReaderBuilder

The `DbReaderBuilder` provides additional configuration options:

### with_checkpoint_id()

Sets the checkpoint ID to use for the reader. If not set, the reader will create and manage its own checkpoint.

```rust
pub fn with_checkpoint_id(mut self, checkpoint_id: uuid::Uuid) -> Self
```

### with_wal_object_store()

Sets a separate object store for the WAL. Use this when the database was configured with a separate WAL object store.

```rust
pub fn with_wal_object_store(mut self, wal_object_store: Arc<dyn ObjectStore>) -> Self
```

### with_options()

Sets the options to use for the reader.

```rust
pub fn with_options(mut self, options: DbReaderOptions) -> Self
```

### with_system_clock()

Sets the system clock to use for the reader.

```rust
pub fn with_system_clock(mut self, system_clock: Arc<dyn SystemClock>) -> Self
```

### with_seed()

Sets the seed to use for the reader's random number generator.

```rust
pub fn with_seed(mut self, seed: u64) -> Self
```

### with_stat_registry()

Sets the stats registry to use for the reader.

```rust
pub fn with_stat_registry(mut self, stat_registry: Arc<StatRegistry>) -> Self
```

### build()

Builds and returns a DbReader instance.

```rust
pub async fn build(self) -> Result<DbReader, Error>
```
